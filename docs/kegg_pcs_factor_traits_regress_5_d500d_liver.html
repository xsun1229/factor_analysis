Introduction

In this part, we inferred 1652 PCs from 332 KEGG pathways. The pipeline is the same with whole blood data. The only differences are: 1.we used liver data in this part, 2.the traits are Total Cholesterol(TC),High Density Lipoprotein(HDL), Low Density Lipoprotein (LDL),Triglycerides(TG)

Effectsize analysis – effect size fitting

For the pairs have more than 2 SNPs with FDR <0.2 (120 pairs), we also tested the effectsize correlation(effect size in GWAS and effect size in factos-association). We fitted the effectsize using: effectsize in GWAS ~ 0 +effectsize in factor-association. We recorded the p-values and r-squareds in the fitting.

To make our results more reliable, we did resampling. We resampled the SNPs without replacement and fitted the points with intercept = 0 again and recorded the r-squareds to compute the p-values in resampling. The ‘p-value from resampling’ is computed by: (number of more extreme values)/(times of resampling).The resampling was repeated 1000 times.

There are 7 pairs show sign consistency (have p-values < 0.05 in resampling, 1.16 fold enrichment). There are 3 pairs at p-value < 0.01, 2.5 fold enrichment).

After deleting the pairs whose ‘num_snp_noMHC’ == 0, we got 119 pairs. We used extended MHC area here (chr6:25726063-33400644). There are 7 pairs show sign consistency (have p-values < 0.05 in resampling, 1.16 fold enrichment). There are 3 pairs at p-value < 0.01, 2.5 fold enrichment).

To summarize the results, We made a scatter plot of all trait-LV pairs. The x-axis shows the R2 in fitting, and the y-axis show the -log10-p values from resampling. To make the plot more informative, Each point represent a trait-LV pair. Different colors show the number of SNPs passing the FDR threshold for each pair.

For the pair have p-value == 0 in resampling, we set the p-value = 0.001 to make the plot
